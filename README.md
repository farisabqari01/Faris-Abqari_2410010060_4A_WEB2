# White Ivy - Channel Management Website

Website Manajemen Channel White Ivy merupakan aplikasi berbasis Laravel 12 yang dikembangkan sebagai proyek Ujian Akhir Semester (UAS) Mata Kuliah Pemrograman Web.

## 🚀 Features

- Authentication (Login & Logout)
- Admin Dashboard
- CRUD Game
- CRUD Streaming Schedule
- CRUD About Channel
- CRUD Social Media
- Dynamic Homepage
- Responsive Design (Bootstrap 5)
- MySQL Database
- Laravel MVC Architecture

## 🛠️ Tech Stack

- Laravel 12
- PHP 8
- Bootstrap 5
- MySQL
- Blade Template
- Eloquent ORM
